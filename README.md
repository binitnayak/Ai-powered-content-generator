# 🚀 Getting Started

```bash
git clone https://github.com/binitnayak/AI-Powered-Content-Generator.git

cd AI-Powered-Content-Generator

npm install

npm run dev
```

---

# 🔑 Environment Variables

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=

CLERK_SECRET_KEY=

DATABASE_URL=

OPENAI_API_KEY=
```

# 🔄 Workflow

```text
User
   │
   ▼
Login with Clerk
   │
   ▼
Choose Template
   │
   ▼
Enter Prompt
   │
   ▼
OpenAI API
   │
   ▼
Generated Content
   │
   ▼
History Saved in Neon DB
```
